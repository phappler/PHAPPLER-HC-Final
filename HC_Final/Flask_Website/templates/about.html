About

Purpose

This website was created as a final project for CHIN 370 - Hacking Chinese Studies. The goal of this project is to run a sentiment analysis on various important texts throughout Chinese history to see how sentiment on various topics has changed over time, as well as providing a good way to display this data.

Sentiment Analysis

Sentiment analysis is a technique that determines how positive or negative a text portrays a certain topic.

Text Selection

As I am not familiar with many texts from Chinese history, the texts I chose were selected from the Timeline of Chinese texts on wikipedia. The texts themselves were pulled from the Project Gutenberg database.

The Algorithm

This algorithm used the NLTK VADER module to sort the input text by which sections have both the highest positive and negative sentiment values.

Citation: Hutto, C.J. & Gilbert, E.E. (2014). VADER: A Parsimonious Rule-based Model for Sentiment Analysis of Social Media Text. Eighth International Conference on Weblogs and Social Media (ICWSM-14). Ann Arbor, MI, June 2014.
